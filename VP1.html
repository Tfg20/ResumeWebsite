Senior Design Project

Project Summary:


For my senior design project I was partnered with one other person and given the task to design a practice firefighting simulation that runs on the hololens and communicates results with a database and website. For my portion of the project I designed function within unity that utilized a post command, which would send information from the hololens to a backend javascript function. That function was hosted on a server connected to the internet so that stats could be sent seemlessly to the MongoDB database. The website that I designed would have the user input a username, created during the simulation, to obtain the information from the database. It would then display them, parsed out, onto the website. I ultilized javascript, nodeJS, HTML, CSS, MongoDB, Unity, and C# to complete my project.

This project was sponsored by the Unites States Navy.
